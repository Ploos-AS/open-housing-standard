# 7. Conformance

An implementation claiming conformance with an OHS standard SHALL satisfy every normative requirement designated by the keyword SHALL.

Recommendations designated by SHOULD are not mandatory but SHOULD be followed unless documented justification exists.

Optional features designated by MAY do not affect conformance.

A conforming implementation SHALL:

- identify the referenced OHS standard and version;
- satisfy every mandatory requirement;
- document any deviations;
- identify optional features implemented;
- provide sufficient documentation for independent verification.
